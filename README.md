# Visualizacion46656 📊

Repositorio dedicado a la gestión y visualización de estructuras de datos complejas. Este espacio centraliza los modelos de nodos y aristas desarrollados para identificar dependencias y cuellos de botella en sistemas de información.

## 📁 Estructura del Proyecto

* **src/**: Scripts de automatización y código fuente.
* **docs/**: Documentación técnica relacionada con auditorías y procedimientos.
* **examples/**: Casos de uso y plantillas de prueba.
* **results/**: Visualizaciones finales (HTML), incluyendo grafos de dependencias bancarias (SQR, SP, tablas).

## 🛠️ Herramientas y Tecnologías

* **Visualización**: Modelado de grafos exportados para análisis en herramientas como Gephi.
* **Automatización**: Scripts `.bat` para la creación de entornos de trabajo estandarizados.
* **Gestión de Datos**: Documentación estructurada de bases de datos y aplicaciones.

## 🚀 Cómo usar este repositorio

1.  **Automatización de carpetas**: Puedes inicializar la estructura ejecutando el archivo `crear_estructura.bat` localmente o mediante el **Workflow** de GitHub Actions diseñado para este repositorio.
2.  **Consulta de Resultados**: Las visualizaciones interactivas se encuentran dentro de la carpeta `/results`.
